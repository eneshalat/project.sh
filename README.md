# project.sh
"./project.sh" is a bootstrapper to create template project files.

## Usage

For creating a C project, use:

```
./project.sh <project-name>
```

For creating a CXX (C++) project, use:

```
IS_PROJECT_CXX=1 ./project.sh <project-name>
```
